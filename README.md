# ecom-analytics-bootcamp

Goal: Build analytics + engineering + ML skills using theLook eCommerce (BigQuery public dataset).

## Setup (Windows / PowerShell)
1. Create venv: `python -m venv .venv`
2. Activate: `.\.venv\Scripts\Activate.ps1`
3. Install: `pip install -r requirements.txt`

- Verified venv + Python + pandas (`src/day1_check.py`)
- Created initial BigQuery exploration queries (`sql/day1_exploration.sql`)
